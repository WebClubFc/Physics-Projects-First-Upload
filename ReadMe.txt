استفاده از لینک محاسبه مقاومت:
1- ابتدا مقادیر مورد نظر را با واحد اهم وارد کنید و با (,) از هم جدا کنید
2- سپس حالت را مشخص کنید که می‌خواهید مجموع مقاومت‌ها را به صورت متوالی محاسبه کنید یا موازی

استفاده از لینک محاسبه شدت جریان الکتریکی:
1- مقدار مورد نظر برای بار الکتریکی را وارد کنید
2- زمان را با واحد ثانیه وارد کنید

استفاده از لینک محاسبه ولتاژ:
1- در آن مقدار بارالکتریکی را با واحد کولن وارد کنید
2- زمان را با واحد ثانیه وارد کنید
3- مجموع مقاومت‌های مقدار را با واحد اهم وارد کنید
